Octagon client — Getting started

Deck creation

  1. This application is highly compatible with Octgn (Windows PC only variant). You may use the very same deck files.
  2. In order to create a new deck, visit CardGameDB Deckbuilder and then export the deck as an Octgn Deck File.

Loading a deck

  1. Drag & drop the deck file onto this very window and you are ready to play.
  2. If that's too complicated, you can always use the old-school dialog:
  • Kneel
  • Stand
  • Turn face down
  • Turn face up
  • Shuffle pile
  • Draw many
  • Browse pile
